OpalDreamCo — Privacy Policy

Effective date: May 15, 2026

1. About this app

PODStudio is a personal analytics and scheduling tool built and operated by OpalDreamCo for use with its own Etsy shop (OpalDreamCo) and its own Pinterest account. It is not distributed to other users, not offered as a service, and not made available to the public.

PODStudio runs locally on the operator's own computer. There is no hosted backend, no shared infrastructure, and no third-party operator.

2. Data PODStudio reads

When OpalDreamCo authorizes PODStudio against an external account, it reads only the data needed to power its own workflows:

From Etsy (via the Etsy Open API v3):

  • OpalDreamCo's own shop listings (title, tags, description, price, lifetime view and favorite counts)
  • OpalDreamCo's own shop receipts and transactions
  • OpalDreamCo's own payment-account ledger entries (Etsy fees, ads spend, refunds)

Etsy scopes requested: transactions_r, listings_r, shops_r — all read-only.

From Printful (via the Printful API):

  • OpalDreamCo's own fulfillment orders and their costs

Printful scopes requested: sync_products:read, orders:read, products:read — all read-only.

From Pinterest (via the Pinterest API v5):

  • OpalDreamCo's own boards (list and metadata)
  • OpalDreamCo's own pins (read)
  • The ability to create new pins on OpalDreamCo's own boards (write)
  • OpalDreamCo's own account profile (username)

Pinterest scopes requested: boards:read, pins:read, pins:write, user_accounts:read.

PODStudio does not read data from any other Etsy shop, Printful account, or Pinterest account.

3. Data PODStudio writes

The only data PODStudio writes to an external service is new pins to Pinterest, on OpalDreamCo's own boards, using content (images, titles, descriptions, links) that OpalDreamCo created or uploaded inside PODStudio.

PODStudio does not modify any Etsy listings, Etsy orders, Printful orders, or Pinterest boards/pins it did not create. PODStudio does not post to any other social network.

4. Where data is stored

All data PODStudio reads is stored in a single SQLite database file on OpalDreamCo's local computer (./data/podstudio.db). Uploaded pin images are stored on the same local computer (./data/uploads/pinterest/).

OAuth access tokens and refresh tokens for all connected services are encrypted at rest using AES-256-GCM with a key held only by OpalDreamCo.

PODStudio does not transmit any of this data to PODStudio's own servers, because PODStudio has no servers. The application communicates directly between OpalDreamCo's computer and the official APIs of Etsy, Printful, and Pinterest.

5. Data sharing

PODStudio does not share, sell, transfer, or disclose any data to any third party. There are no advertisers, no analytics vendors, no error-tracking services, and no cloud backups.

6. Data retention and deletion

Because all data lives on OpalDreamCo's local computer, the operator can delete it at any time by removing the ./data/ directory.

OAuth connections can be revoked at any time:

  • Etsy: https://www.etsy.com/your/apps
  • Printful: https://www.printful.com/oauth/tokens
  • Pinterest: https://www.pinterest.com/settings/apps

Revoking a connection from the provider's side immediately stops PODStudio's access.

7. Children's data

PODStudio is not directed to children and does not knowingly collect any data from children.

8. Security

  • OAuth flows use industry-standard authorization-code with PKCE.
  • All tokens are encrypted at rest (AES-256-GCM) before being written to disk.
  • The application binds to http://localhost:3000 on OpalDreamCo's own machine and is not exposed to the public internet.

9. Changes to this policy

If material changes are made to how PODStudio handles data, this document will be updated and the "Effective date" above will be revised.
